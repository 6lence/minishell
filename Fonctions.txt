Echo:

Command: echo Hello, World!
Expected Output: Hello, World!
Change Directory (cd):

Command: cd [directory_name]
You can use pwd to print the working directory and verify the change.
Print Working Directory (pwd):

Command: pwd
Expected Output: The current working directory path.
Environment Variables (env):

Command: env
Expected Output: A list of environment variables.
Export:

Command: export TEST_VAR=hello
Use env to see if TEST_VAR has been set.
Use echo $TEST_VAR to see if it prints hello.
Unset:

Command: unset TEST_VAR
Use env to see if TEST_VAR has been removed.
Exit:

Command: exit
Expected: The shell should exit.

-------

minishell-> echo Hello, World!
Hello, World!

minishell-> pwd
/path/to/current/directory

minishell-> cd /
minishell-> pwd
/

minishell-> env
... (list of environment variables) ...

minishell-> export TEST_VAR=hello
minishell-> echo $TEST_VAR
hello

minishell-> unset TEST_VAR
minishell-> echo $TEST_VAR

minishell-> cd non_existent_directory
cd: non_existent_directory: No such file or directory

minishell-> exit
